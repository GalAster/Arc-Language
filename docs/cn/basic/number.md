# Number


## allow


### integer

整数默认是大整数类型


### typed integer

整数的注解在后方

不能带空格

```bash
255u8
-127i8
```

### decimal



### bytes

```
0x0
0o0
0f0
```



## warning

- 不要使用多重下划线

```bash
1__2
```

- 不要使用多重前导零

```bash
0000
0001
00.2
```

- 不要使用大小写混用的比特输入

```bash
0xFfF
```




## deny

- 禁止残缺小数

```bash
123.
.012
```

- 禁止残缺下划线

```bash
1_
_12
0._5
```

- 禁止多重符号

```bash
+-1
++0
```

- 禁止多重小数
```bash
1.0.0
```

- 禁止多重字节

```bash
0x
```
