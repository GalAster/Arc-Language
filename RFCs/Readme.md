Arc's Request for Comments
==========================

## States

- 已接受: 该特性已被接受并生效(✔️).
- 可追加: 该特性已被接受且可追加条例(➕).
- 可选的: 该特性是否实现由作者自行决定(❕).
- 讨论中: 该特性正在讨论细化中(❔).
- 已否决: 该特性已被否决或覆盖(❌).
